# 🛒 Online Shopping System (Java)

##  Overview
This is a console-based Online Shopping System developed using Java.  
The project simulates a basic e-commerce platform where users can browse products, add items to cart, and place orders.

---

##  Features
- Product browsing
- Add to cart functionality
-  Update product quantity in cart
-  Total bill calculation
-  Order processing
-  Admin product management (optional)
-  Basic exception handling

---

## Concepts Used
- Object-Oriented Programming (OOP)
  - Encapsulation
  - Abstraction
  - Polymorphism
- Collections (ArrayList)
- File Handling (if used)

---

##  Project Structure
- `Product.java` → Stores product details  
- `Cart.java` → Manages cart operations  
- `Main.java` → Entry point of application  

---

##  How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/online-shopping-system.git
2. Open the project in VS Code
3. Compile the code:
   javac Main.java
4. Run the program:
   java Main
